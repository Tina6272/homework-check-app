# 首頁名稱與風格備選

本資料夾集中保存 Tina 指定保留的 v2 三版，以及名稱 1「學會自己來」的新風格三版。所有檔案均為比較與定稿輸入，不代表正式首頁或 Gate 3 已由 Tina 核准。

## 共同內容

- App 暫用名稱：`學會自己來`
- 已選副標題：`從每天的作業開始，一步一步學會自己來。`
- 三角色入口：`我是學生`、`我是老師`、`我是家長`
- 畫面原則：乾淨、有質感、專業；不放星星名稱、設計標籤、隱私小字或其他功能說明。

## v2 入選三版

位於 `v2入選三版/`，保留可直接開啟的 HTML 與共用 `shared.css`、`shared.js`：

1. v2-02 學習護照／手帳
2. v2-03 課後班溫暖教室
3. v2-06 自然探險／成長島

`v2畫面截圖/` 另存同三版的桌面畫面，供小娜與 Tina 快速比較。C 槽原始 v2 檔案未修改。

## 名稱 1 新風格三版

位於 `名稱1-學會自己來-新風格三版/`：

1. 風格 1「溫暖連結」：安心、陪伴、三方各有位置；融合「點點各有去處」與「不同作品互相回應」。
2. 風格 2「自由變化」：鮮明、可塑、保留創意；融合「形狀正在變」與「這次這樣，下次也能不同」。
3. 風格 3「開放探索」：寬廣、成熟、保留未知；融合「路可以分開走」與「留一塊還不知道」。

## 下一個停止點

下一階段先由 Tina 與小娜完成「名稱＋副標題＋風格」定稿，再規劃通用首頁。定稿前不製作學生端、老師端、師生連動、家長端或正式功能，也不把任何一版視為最終選定。
