## 下一階段要實作的老師端方案

討論順序已調整為：老師介面、學生介面、家長介面。下一階段先實作老師介面。

老師端主流程：

1. 採「逐班輸入、逐班派發」：先輸入 101 班作業並派發，再輸入 102 班，再輸入 103 班。
2. 每次派發是一個「單班單日」批次，包含班級、日期、座號範圍、出席/請假、學生狀態與多筆作業。
3. 派發對象用座號範圍，不輸入學生姓名。
4. 派發後停在結果畫面，老師可檢查當班成果，再按「新增下一班」繼續。

老師端輸入欄位：

1. 班級：自由輸入，例如 `101 班`、`102 班`、`103 班`。
2. 日期：預設今天，可修改。
3. 座號範圍：起始座號、結束座號，例如 1 到 10。
4. 出席人數、請假人數：可由座號格統計，也可顯示給老師確認。
5. 學生座號格：每個座號可勾選 `請假`、`參加社團`。

學生座號狀態規則：

1. 請假學生仍產生作業任務，但老師端不追蹤完成進度。
2. 請假學生座號格要反白，讓老師一眼知道不用追蹤當日完成度。
3. 家長端後續要能看到請假當天仍需完成的作業內容，將監督責任交棒給家長。
4. 參加社團學生仍追蹤完成進度，但座號格要標示 `社團`，讓老師知道進度慢可能是參加社團所致，也方便老師與家長重新評估。

作業模板規則：

1. 作業種類下拉包含 18 種：課本、習作、作業簿(翰林版)、作業簿(南一版)、練習簿(康軒版)、重點複習(康軒版)、隨堂演練(南一版)、課堂練習(翰林版)、國語甲本、國語乙本、國語語詞簿、國語造句簿、考試簿、測驗卷、測驗卷訂正、閱讀分享單、閱讀護照、其他。
2. 有版本的作業顯示版本下拉：康軒版、翰林版、南一版。
3. 有科目的作業顯示科目下拉：依類型顯示國語、數學、英文、生活、社會、自然。
4. 範圍使用混合格式：課次/單元、頁碼、題號、補充說明。
5. 每筆作業有建議分鐘，可調整。

當日儀錶板規則：

1. 儀錶板依班級分組，顯示當天所有班級的作業內容與派發成果。
2. 不只顯示各階段總人數；每位學生座號旁都要有即時、明顯的紀錄。
3. 每個座號旁顯示三色比例條：綠色為已完成、灰色為待完成、紅色為需協助。
4. 比例以該學生當天派發作業數為分母。
5. 座號旁同時顯示簡短數字，例如 `完成 2 / 待 1 / 協助 1`。
6. 需要協助的任務，老師實體協助後可勾選為 `已協助完成`。
7. `已協助完成` 不再列入需協助，但要與學生自行完成區分。
8. 派發錯誤時可即時修正；修正後只呈現最新正確結果，不保留錯誤資訊。
