AI Agent 工具建構課後班 App 的可行性、費用與路徑比較

能否用 AI Agent 完成這款 App？
網頁版（Web App）── 完全可行。Bolt.new、Lovable、Replit Agent 均可從文字描述直接產出含資料庫、登入、即時同步的完整 Web App，老師端 Dashboard 與家長端通知皆可在網頁實現。
iOS / Android 原生 App── 有條件可行。需使用 FlutterFlow 或 Adalo，AI 輔助建構，但需手動設定後端與推播通知，發布至 App Store 需額外費用（Apple 開發者帳號 $99/年）。
AI 辨識作業圖片── 可整合，但需額外 API。Google Vision API 或 Mathpix 可辨識手寫作業，免費額度有限，大量使用需付費。
零費用完成「完整成品」── 不現實。免費工具可做出原型與 MVP，但正式上線（自訂網域、多班級資料、推播通知）仍需基本月費。
主要 AI Agent 工具比較（2026）
Bolt.new
網頁 最慷慨免費
從文字提示直接產出 React + 資料庫全端 Web App，無需任何程式知識。
免費：每月 100 萬 tokens
付費：$25/月（1,000 萬 tokens）
Lovable
網頁 介面最精美
整合 Supabase 後端，UI 最接近商業產品水準，含 GitHub 同步。
免費：每月 30 次（5/天）
付費：$25/月（100 次 credits）
Replit Agent
網頁 最自主
30+ 服務整合，可自動測試、偵錯、部署，適合較複雜邏輯。
免費：每日少量 Agent 額度
付費：$20/月（含 $25 使用額度）
FlutterFlow
原生 App iOS／Android
可發布至 Apple Store 與 Google Play，需配合 Firebase 後端。有視覺化拖拉介面。
免費：可建構但不能上架
付費：$39/月（Basic，可發布）
Adalo
原生 App 最易上手
號稱「像 PowerPoint 一樣簡單」，含內建資料庫，可一鍵發布 iOS ／ Android。
免費：可預覽，不能上架
付費：$45/月（含上架功能）
Claude Code / Cursor
全端 開發者專用
需有基本程式概念，自由度最高，適合需要客製化複雜功能的情境。
Cursor：$20/月
Claude Code：需 Claude Pro/Max
費用路徑：兩條方向
路徑 A ── 網頁版 MVP（推薦起點）
$0 ～ $25／月
Bolt.new 免費版產出原型 → 付費 $25/月 正式上線
老師端、家長端均為網頁，學生端可加入 PWA（可存在手機桌面）
AI 辨識使用 Google Vision API 免費額度（每月 1,000 次）
路徑 B ── 原生 App（完整成品）
$64 ～ $150／月
FlutterFlow $39/月 + Firebase $25/月 起跳
Apple 開發者帳號 $99/年（另計）
適合確認需求穩定後、準備正式推廣時採用
建議路線
第一階段：用 Bolt.new 免費版建立網頁原型，在 1-2 個班級試用，驗證五步驟流程是否真正減輕老師負擔。這個階段費用為 $0。

第二階段：確認流程有效後，升級 Bolt.new（$25/月）或改用 FlutterFlow（$39/月）做原生 App，申請 Apple 開發者帳號，正式上架。月費約 $25 ～ $64。

注意：「免費就能完成成品」是市場常見的誤導。免費工具可以完成 原型與內部測試版，但凡涉及自訂網域、多用戶即時資料、推播通知、App Store 上架，都需要基本月費。
